## 简介

`astrbot_plugin_eatdrink` 是一个随机推荐吃喝什么的插件。

## 功能&教程

- **随机推荐吃的**：发送 /吃什么 推荐吃的 吃点啥 进行随机推荐吃的
- **随机推荐喝的**：发送 /喝什么 推荐喝的 喝点啥 进行随机推荐喝的
- **一起推荐吧！**：发送 /吃喝什么 随机推荐吃喝汇总！
