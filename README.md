# astrbot\_plugin\_eatdrink

**[用于随机推荐吃什么 喝什么 或是 吃喝什么一起] 食物推荐插件**

</div>

## 简介

`astrbot_plugin_eatdrink` 是一个随机推荐吃喝什么的插件。

## 功能&教程

- **随机推荐吃的**：
    发送 "/吃什么" 推荐吃的 吃点啥 进行随机推荐吃的
    发送 "/吃什么 烧烤" 推荐吃什么烧烤，类似的还有火锅、主食、肉菜、日料、披萨等
- **随机推荐喝的**：
    发送 "/喝什么" 推荐喝的 喝点啥 进行随机推荐喝的
    发送 "/喝什么 奶茶" 推荐喝什么奶茶，类似的还有咖啡、果汁、气泡水、甜品等
- **一起推荐吧！**：
    发送 "/吃喝什么" 随机推荐吃喝！

吃喝分类预设模板在第一次安装插件时会复制到/plugin_data/astrbot/plugin/eatdrink/下，可根据个人喜好添加
添加时记得在名称后面加上"|"管道符号来进行详细分类以启用分类推荐
